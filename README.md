# CertTrack

CertTrack is an internal web-based application developed during industrial internship (PKL).
The system is designed to monitor operator certificates and provide early warnings before expiration.

## Features
- Role-based access (Admin & Operator)
- Certificate validity monitoring
- Expiration warning (H-14)
- Lightweight UI for internal factory environment

## Tech Stack
- PHP
- HTML
- CSS
- MySQL
